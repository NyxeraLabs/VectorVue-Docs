# Privacy Policy (Future SaaS Version)

## 1. Data Collection

VectorVue SaaS may collect:

- Account identifiers
- Authentication logs
- Operational metadata
- System telemetry

## 2. Data Usage

Data is used for:

- Security monitoring
- Platform integrity
- Abuse prevention
- Compliance enforcement

## 3. Data Storage

- Logs may be retained for compliance
- Sensitive operational data remains tenant-isolated
- Encryption at rest and in transit enforced

## 4. No Sale of Data

NyxeraLabs does not sell user data.

## 5. Regulatory Alignment

Future SaaS version aims to align with:

- GDPR
- SOC 2
- ISO 27001 controls
